HTML Favicon Tutorial

Look at the browser tab to see the favicon next to the page title.

What is a Favicon?

A favicon is a small image displayed next to the page title in the browser tab.

How to Add a Favicon

  1. Create or obtain a favicon image (common name: favicon.ico)
  2. Save the favicon to your website's root directory or in an images folder
  3. Add the link element in the head section of your HTML file

Supported File Formats

Modern browsers support various file formats for favicons:

  • ICO
  • PNG
  • GIF
  • JPEG
  • SVG
