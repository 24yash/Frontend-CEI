CSS Border Properties

Property Description
border Shorthand for all border properties
border-style Sets border style (required for visibility)
border-width Controls border thickness
border-color Sets border color (inherits text color if unset)
border-radius Creates rounded corners
border-[top|right|bottom|left] Individual side borders
