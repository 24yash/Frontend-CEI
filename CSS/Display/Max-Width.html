CSS Width and Max-Width Example

Using width

This div has a width of 500px. If the browser window is smaller than 500px, a horizontal scrollbar will appear.

Using max-width

This div has a max-width of 500px. If the browser window is smaller than 500px, it will adjust to the window size without creating a horizontal scrollbar.

Note: Resize the browser window to see how the two divs behave differently.
